#### 🎃 Docker部署

1. 安装Docker
2. 打开终端并进入brisk目录
3. 执行以下命令，安静等待brisk启动即可（docker-compose 一键部署。）


```bash
docker-compose -f docker-compose.yaml up
```

#### 🎉 二次开发

1. 拉取代码

```bash
$ git clone https://gitee.com/wieszheng/brisk.git
$ cd brisk
```

2. 安装依赖

```bash
# 可更换豆瓣源或者清华源安装依赖
$ pip install -r requirements.txt
```

3. 安装并启动mysql


4. 修改conf/.env


5. 启动服务

```bash
$ python brisk.py
```
